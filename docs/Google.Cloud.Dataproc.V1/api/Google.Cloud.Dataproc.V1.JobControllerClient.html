ShutdownDefaultChannelsAsync()

Shuts down any channels automatically created by Create(ServiceEndpoint, JobControllerSettings) and CreateAsync(ServiceEndpoint, JobControllerSettings). Channels which weren't automatically created are not affected.

Declaration
public static Task ShutdownDefaultChannelsAsync()
Returns
Type Description
System.Threading.Tasks.Task

A task representing the asynchronous shutdown operation.
Remarks

After calling this method, further calls to Create(ServiceEndpoint, JobControllerSettings) and CreateAsync(ServiceEndpoint, JobControllerSettings) will create new channels, which could in turn be shut down by another call to this method.
