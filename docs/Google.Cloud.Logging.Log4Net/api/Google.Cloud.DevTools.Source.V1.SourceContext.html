Class SourceContext

A SourceContext is a reference to a tree of files. A SourceContext together with a path point to a unique revision of a single file or directory.

GitRevisionIdLogLabel

Gets the custom log label of Stackdriver Logging entry to set Git revision id. When writing Stackdriver Logging entry, user should add this custom label, set the value to the git revision id. This enables Google Cloud Tools for Visual Studio to locate source file revision of the log entry. The design is still evolving, the usage of the label is subject to change.
