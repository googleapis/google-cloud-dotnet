Google.Cloud.Spanner.Data

Google.Cloud.Spanner.Data is the ADO.NET provider for Google Cloud Spanner. It is the recommended package for regular Spanner database access from .NET.

The Google.Cloud.Spanner.Admin.Database.V1 and Google.Cloud.Spanner.Admin.Instance.V1 packages should be used for database and instance administration respectively, such as creating or deleting databases and instances.

Installation

Install the Google.Cloud.Spanner.Data package from NuGet. Add it to your project in the normal way (for example by right-clicking on the project in Visual Studio and choosing "Manage NuGet Packages..."). Please ensure you enable pre-release packages(for example, in the Visual Studio NuGet user interface, check the "Include prerelease" box).
