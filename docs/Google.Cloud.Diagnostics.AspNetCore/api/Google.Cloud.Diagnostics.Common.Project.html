GetAndCheckProjectId(String, MonitoredResource)

Determines the correct project id from a string project id and a Google.Api.MonitoredResource. If the specified project id is not null, it is returned. Otherwise, if the project id of the MonitoredResource is not null, it is returned. It both are null, an System.InvalidOperationException is thrown.
