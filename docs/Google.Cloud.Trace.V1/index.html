Google.Cloud.Trace.V1

Google.Cloud.Trace.V1 is a .NET client library for the Stackdriver Trace API.

Installation

Install the Google.Cloud.Trace.V1 package from NuGet. Add it to your project in the normal way (for example by right-clicking on the project in Visual Studio and choosing "Manage NuGet Packages..."). Please ensure you enable pre-release packages (for example, in the Visual Studio NuGet user interface, check the "Include prerelease" box).

Authentication

To authenticate all your API calls, first install and setup the Google Cloud SDK. After that is installed, run the following command in a Google Cloud SDK Shell:

> gcloud auth application-default login

Getting started

The main entry point for the API is TraceServiceClient. Create an instance using the static Create method, optionally specifying a service endpoint and settings.

Sample code

List traces

TraceServiceClient client = TraceServiceClient.Create();
PagedEnumerable<ListTracesResponse, Trace> traces = client.ListTraces(projectId);
foreach (Trace trace in traces.Take(10))
{
    Console.WriteLine($"{trace.TraceId}:");
    foreach (TraceSpan span in trace.Spans)
    {
        Console.WriteLine($"  {span.StartTime}-{span.EndTime}: {span.Kind}");
    }
}
