Syntax
public sealed class StorageClientImpl : StorageClient
Remarks

This is the "default" implementation of StorageClient. Most client code should refer to StorageClient, creating instances with

Create(GoogleCredential) and

CreateAsync(GoogleCredential). The constructor of this class is public for the sake of constructor-based dependency injection.
