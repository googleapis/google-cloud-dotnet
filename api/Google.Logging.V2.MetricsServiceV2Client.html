ShutdownDefaultChannelsAsync()

Shuts down any channels automatically created by Create(ServiceEndpoint, MetricsServiceV2Settings) and CreateAsync(ServiceEndpoint, MetricsServiceV2Settings). Channels which weren't automatically created are not affected.

Remarks

After calling this method, further calls to Create(ServiceEndpoint, MetricsServiceV2Settings) and CreateAsync(ServiceEndpoint, MetricsServiceV2Settings) will create new channels, which could in turn be shut down by another call to this method.
