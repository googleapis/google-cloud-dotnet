public sealed class DatastoreDbImpl : DatastoreDb
Remarks

This is the "default" implementation of DatastoreDb. Most client code should refer to DatastoreDb, creating instances with

Create(String, String, DatastoreClient). The constructor of this class is public for the sake of constructor-based dependency injection.
